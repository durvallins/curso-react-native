criar projeto react-native
npx react-native init NomeDoProjeto

entrar na pasta do projeto criado
cd NomeDoProjeto

iniciar instalação (já com o avd aberto)
npx react-native run-android
espere a conclusão...

neste projeto será instalado dependências do react-native navigation
https://reactnavigation.org/docs/getting-started/

execular:
npm install @react-navigation/native
npm install react-native-screens react-native-safe-area-context

ao final da página web, clicar em: Next (Hello React Navigation) >>

instalar a navegação steck (pilhas)
npm install @react-navigation/native-stack

no lado esquerdo da pagina, procurar GUIDES > Tab navigation
npm install @react-navigation/bottom-tabs

Voltar a Guides, e clicar em Drawer navigation e na pagina q abrir, 
clicar no link  installation instructions, pra abrir o texto e poder pegar o comando:
npm install @react-navigation/drawer

neste projeto, serão instaladas as depend. já citadas. 

após instalar tudo, executar:

npx react-native run-android
